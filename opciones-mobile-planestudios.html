📱 10 Opciones de Layout Mobile

⚠️ Nota: Todas estas opciones mejoran la experiencia en celular. Elige la que mejor se adapte a tu marca y flujo de usuario.
OPCIÓN 1 - RECOMENDADA
📋 Tabs Deslizables (Scroll Horizontal)
Los botones de semestres ocupan el ancho y se pueden deslizar horizontalmente si no caben todos.
Vista mobile:
1º sem
2º sem
3º sem
4º sem
5º sem
...
↑ Se pueden deslizar con el dedo
✓ Ventajas:
Usa todo el espacio horizontal
Muy intuitivo en mobile
Materias y leyenda visibles abajo
Scroll suave y natural
⚠ Desventajas:
Requiere CSS overflow-x
MI FAVORITA - Mejor experiencia
OPCIÓN 2 - RECOMENDADA
🎯 Dropdown/Select HTML Nativo
Un selector <select> limpio. Al seleccionar, se despliega el contenido abajo.
Vista mobile:
↓ Se abre el selector nativo del teléfono
✓ Ventajas:
Muy compacto (ocupa poco espacio)
Selector nativo = mejor entra experiencia
Sin JavaScript complejo
Accesible (WCAG)
⚠ Desventajas:
Menos visual que botones
Styling limitado
SEGUNDA MEJOR - Muy práctica
OPCIÓN 3
🎠 Carrusel/Swiper de Semestres
Los semestres se comportan como un carrusel, puedes deslizar de uno a otro con transiciones suaves.
✓ Ventajas:
Animaciones suaves y modernas
Cambio de semestre muy fluido
Indicador visual claro
⚠ Desventajas:
Requiere librería (Swiper, Slick)
Mayor consumo de JS
OPCIÓN 4
🆙 Leyenda Arriba - Materias Debajo
Cambiar el orden: mostrar leyenda primero (pequeña y compacta), luego materias.
✓ Ventajas:
Leyenda visible antes de bajar
Estructura clara
⚠ Desventajas:
Leyenda puede ser repetitiva
OPCIÓN 5
📌 Leyenda Flotante (Sticky/Fixed)
La leyenda se queda fija en la parte baja de la pantalla mientras scrolleas materias.
✓ Ventajas:
Leyenda siempre visible
No requiere scroll para ver referencia
⚠ Desventajas:
Ocupa espacio permanente
Puede ocultar last materia
OPCIÓN 6
🔘 Botones Pequeños - 3 por Fila
Los 7 botones de semestre en grid 3-3-1, más pequeños pero visibles todos a la vez.
1
2
3
4
5
6
✓ Ventajas:
Todos los semestres visibles
Sin scroll horizontal
⚠ Desventajas:
Botones muy pequeños
Difícil de clickear en móvil
OPCIÓN 7
🎚️ Según Leyenda - Toggle Tabs
Agrupar semestres por tipo de contenido: "Teóricos" (1,2,4,6) vs "Prácticas" (3,5,7) con tabs.
✓ Ventajas:
Agrupación lógica
Menos botones visibles
⚠ Desventajas:
Requiere 2 clics para cambiar
Puede confundir
OPCIÓN 8
📱 Acordeón (Accordion)
Cada semestre es un acordeón que se expande/colapsa para mostrar materias. Leyenda expandible aparte.
✓ Ventajas:
Muy compacto inicialmente
Fácil de navegar
⚠ Desventajas:
Solo ves un semestre a la vez
Más clics para comparar
OPCIÓN 9
❓ Leyenda Modal/Info Flotante
Un ícono "i" que abre una ventana modal con la leyenda. Las materias ocupan toda la pantalla.
✓ Ventajas:
Máximo espacio para materias
Leyenda sin desorden
⚠ Desventajas:
Requiere clic extra para ver leyenda
OPCIÓN 10 - RECOMENDADA
🎯 Híbrida: Select + Scroll Horizontal Compacto
Dropdown para cambiar + tabs scrollables pequeños como referencia visual. Leyenda colapsable abajo.
✓ Ventajas:
Mejor de ambos mundos
Compacto pero visual
Leyenda no molesta
Accesible y moderno
⚠ Desventajas:
Ligeramente más complejo
TERCERA MEJOR - Completa y balanceada

Resumen de recomendaciones:

🥇 Opción 1: Tabs deslizables (mejor UX)

🥈 Opción 2: Select HTML (más simple)

🥉 Opción 10: Híbrida (mejor balance)
